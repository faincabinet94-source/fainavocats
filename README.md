# Ultra-Premium Dark Mode Landing Page

A stunning, minimalist landing page with dark mode SaaS aesthetic inspired by Linear, Vercel, and Aceternity.

## Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Animations
- **Lucide React** - Icons

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- ✨ Ultra-premium dark mode design
- 🎨 Glassmorphism effects
- ⚡ Lightning-fast animations
- 🌐 Fully responsive
- 🎯 SEO optimized
- 🔒 TypeScript for type safety

## Design System

- **Background**: `#020202` with subtle noise texture
- **Accent Colors**: Electric Blue (`#3b82f6`) and Purple (`#8b5cf6`)
- **Typography**: Inter font with tight tracking
- **Animations**: Smooth 700ms transitions with ease-out
